# Appointment-booking-system

Files:
- index.html
- style.css
- script.js

Execution:
1. Extract the ZIP file.
2. Open the project folder.
3. Double-click `index.html` OR open it in Chrome/Edge.
4. Enter Name, Date and Time.
5. Click **Book**.
6. Bookings are stored in browser Local Storage.

Mini Project Description:
A simple front-end appointment booking system using HTML, CSS and JavaScript. It allows users to book appointments and displays all bookings on the page.
